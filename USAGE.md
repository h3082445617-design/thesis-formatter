# 使用说明 - Usage Guide

## 快速开始

### 方式 1：在线使用（推荐）
1. 打开网址
2. 拖拽或点击选择你的 .docx 论文文件
3. 点击"开始排版"
4. 等待处理（通常 3-10 秒）
5. 自动下载排好版的文件

### 方式 2：本地开发
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py

# 打开浏览器访问
http://localhost:5000
```

## 支持的格式

- **输入：** .docx 格式（现代 Word 文档）
- **输出：** .docx 格式（格式化后）
- **文件大小：** 最大 50MB
- **处理时间：** 通常 3-10 秒（50-100页论文）

## 格式化内容

本工具自动应用以下国标 GB/T 7714 格式化：

| 元素 | 设置 |
|------|------|
| 正文字体 | 宋体 12pt |
| 行距 | 1.5倍 |
| 一级标题 | 黑体 18pt 加粗 居中 |
| 二级标题 | 黑体 14pt 加粗 左对齐 |
| 三级标题 | 宋体 12pt 加粗 左对齐 |
| 页码 | 页脚中心 阿拉伯数字 |
| 页边距 | 左 2.5cm 右 2cm 上下 2cm |
| 段落间距 | 段前 0pt 段后 6pt |

## 常见问题

### Q: 为什么无法上传我的文件？
A: 请检查：
- 文件格式是否为 .docx（不支持 .doc、.odt 等）
- 文件大小是否不超过 50MB
- 文件是否损坏（在 Word 中试试能否打开）

### Q: 排版后还需要手动调整吗？
A: 正常情况下不需要。但如果论文有特殊格式或不兼容的样式，可能需要手动微调。

### Q: 我的文件会被保存吗？
A: 不会。文件上传→处理→下载后立即删除，不保存在服务器。完全隐私保护。

### Q: 可以多次上传和处理吗？
A: 可以。直接再次上传即可。

### Q: 处理失败了怎么办？
A: 尝试以下步骤：
1. 在 Word 中用"修复"功能检查文件
2. 删除文件中可能的冲突样式
3. 重新上传重试
